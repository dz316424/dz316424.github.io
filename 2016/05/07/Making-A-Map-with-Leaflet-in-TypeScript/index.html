Making Map with Leaflet in TypeScript

May 07, 2016 发布在 GIS

Recently I am working on a web mapping project based on Angular 2 , an increasingly popular succeeding framework of Angular. In this post, I will briefly talk about how to make a map with Leaflet in TypeScript, whichi is also used in Angular 2.

Preparation

I assume you have had some basic understanding on TypeScript. If not, the official documentation is an excellent start. (if you are a former .Net developer, you must feel familiar - it’s basically a js-like C#)

Also, since TypeScript is still a developing language proposed by Microsoft, mainstream browsers don’t provide support to TypeScript directly. To run a web project in TypeScript, you will probably need to compile it into JavaScript. There are several choice of compiler for selection, but that’s out of the scope of this post. If you are also interested in Angular 2, the 5-minutues Angular 2 tutorial provides a simple but fully functional setup.

Type Declaration
